# Bug Reports

This directory contains documented bugs discovered during manual and API
testing of web applications.

Each bug report includes: - Steps to reproduce - Expected vs actual
result - Environment details - Evidence (screenshots)

All screenshots are stored in the `/Evidence` directory and linked
inside the reports.
