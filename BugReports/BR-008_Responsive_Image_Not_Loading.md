# BR-008 — Responsive Image Does Not Load on Mobile Layout

**Severity:** Medium  
**Priority:** Medium  
**Status:** Open  
**Component:** Responsive Layout / Images

---

## Summary
Hero image fails to load on mobile resolution.

---

## Description
At widths below 768px, hero banner image either does not load or loads partially.
This breaks layout composition and visual presentation.

---

## Steps to Reproduce
1. Open site in responsive mode.
2. Set width to 360–768px.
3. Refresh page.

---

## Expected Result
Hero image loads correctly at all breakpoints.

---

## Actual Result
Blank placeholder or broken image icon.
