# HTTP 418 I'm a teapot

**Цель:** Зафиксировать нестандартный статус-код 418 и поведение Fetch.

## Запрос
- **Endpoint:** `https://httpbin.org/status/418`
- **Инструмент:** Chrome DevTools → Console / Network
- **Метод:** см. скриншоты (Console/Network)

## Ожидалось
Status = 418, возможна запись об ошибке в консоли.

## Фактически
В консоли: GET ... 418 (I'm a teapot) и Status: 418.

## Доказательства
Скриншоты в папке `evidence/`.
