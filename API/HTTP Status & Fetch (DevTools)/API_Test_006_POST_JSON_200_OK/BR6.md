# API Test — POST JSON Request (200 OK)

## Purpose
Send a POST request with a JSON payload and verify request/response data in DevTools.

## Request Details
- **Endpoint:** https://httpbin.org/post
- **Tool:** Chrome DevTools (Console / Network)
- **Method:** POST
- **Payload:** JSON

## Expected Result
- Response status: **200 OK**
- Request payload contains submitted JSON
- Response echoes submitted data.

## Actual Result
- Status Code: **200 OK**
- Network panel shows request payload and returned JSON data.
